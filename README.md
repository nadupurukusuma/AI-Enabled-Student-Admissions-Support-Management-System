# AI-Enabled-Student-Admissions-Support-Management-System
A Salesforce CRM application that streamlines student admissions, enrollment management, approval workflows, and AI-assisted support using Agentforce.

## 📌 Overview

The AI-Enabled Student Admissions & Support Management System is a Salesforce CRM application developed to automate and simplify the student admission process. It centralizes student records, course management, enrollment tracking, and approval workflows while leveraging Agentforce AI to support administrative decision-making.

## 🚀 Features

* Student Registration and Profile Management
* Course Management
* Student Enrollment Tracking
* Automated Approval Workflows
* AI-Assisted Application Summary using Agentforce
* Enrollment Risk Analysis
* Reports and Dashboards
* Role-Based Access Control
* Validation Rules and Flow Automation

## 🛠️ Technologies Used
